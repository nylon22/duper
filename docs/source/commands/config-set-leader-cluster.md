## Usage

```sh
duper config set-leader-cluster [options]
```

## Options

| Option | Description | Type | Alias | Required |
| -------- | ----------- | ------- | ------- | -------- |
| `name` | The friendly name for the Elasticsearch cluster | `string` | `n` | **True** |
